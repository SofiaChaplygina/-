Вариант 2. Реализуйте рекурсивную функцию для вычисления степени числа (power(x, n)).

Алгоритм вычисления степени числа:
Основная идея алгоритма заключается в использовании рекурсии для эффективного вычисления степени числа. Вместо простого умножения числа x на себя n раз, мы используем свойство:
Если n четное: xⁿ = (x²)^(n/2)
Если n нечетное: xⁿ = x * (x²)^((n-1)/2)
Это позволяет сократить количество операций.

Пошаговое описание алгоритма:
Базовый случай: если n = 0, вернуть 1 (любое число в степени 0 равно 1)
Если n отрицательное, вычислить 1 / power(x, -n)
Если n четное:
Вычислить рекурсивно power(x*x, n/2)
Если n нечетное:
Вычислить x * power(x*x, (n-1)/2)

Временная сложность: O(log n)
Объяснение: На каждом шаге мы уменьшаем показатель степени вдвое
Количество рекурсивных вызовов пропорционально log₂n
На каждом уровне рекурсии выполняется постоянное количество операций
Это значительно эффективнее, чем наивный подход с временной сложностью O(n), где мы умножаем число на себя n раз.
Пространственная сложность: O(log n) из-за рекурсивного стека вызовов.

Ответ на вопрос №9:
Перестановки и сочетания — это два ключевых понятия комбинаторики, которые различаются главным образом отношением к порядку элементов. Перестановки представляют собой упорядоченные комбинации элементов, 
где важен порядок их расположения, и изменение порядка создаёт новую перестановку. Сочетания, напротив, являются неупорядоченными комбинациями, где важен только состав элементов, а не их расположение.
